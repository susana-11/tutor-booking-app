## ðŸ“Š State Management

### Reply State
```dart
// State variables
Message? _replyingTo;  // null = no reply

// Set reply
setState(() {
  _replyingTo = message;
});

// Clear reply
setState(() {
  _replyingTo = null;
});

// Send with reply
sendMessage(
  content: content,
  replyToId: _replyingTo?.id,
);
```

### Forward State
```dart
// No persistent state needed
// Uses temporary variables in methods

// Show forward sheet
showModalBottomSheet(
  builder: (context) => ForwardSheet(message),
);

// Forward message
await chatService.sendMessage(
  conversationId: targetId,
  content: message.content,
  type: message.type,
  attachments: message.attachments,
);
```
